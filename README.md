# Denmark

🔗 **[Live Demo](https://phonebook-alpha.vercel.app/)**

## Tech Stack

- **Core:** React, Vite
- **State Management:** Redux Toolkit, Redux Persist
- **Styling:** Styled Components
- **Animations:** Framer Motion
- **Navigation:** React Router Dom
- **UI Components:** React Icons, React Loader Spinner
- **Media Handling:** Yet Another React Lightbox, React Zoom Pan Pinch
- **Networking:** Axios

## Getting Started

### 1. **Clone the repository:**

```bash
   git clone https://github.com/Dmytro1117/denmark_react.git
```

### 2. **Install dependencies:**

```bash
   npm install
```

### 3. **Configure Environment Variables:**

Create a `.env` file in the root directory and add your backend connection and API-Weather:

```env
VITE_BASE_BACKEND_URL=https://denmark-api-node-js.onrender.com
```

```env
VITE_WEATHER_KEY=your_key_from_https://api.openweathermap.org/geo/1.0
```

### 4. **Run the application:**

```bash
npm run start
```

The app will be available at http://localhost:5173/
